# Getting Started

Welcome to the Getting Started guide! This section will help you set up and configure our project for the first time.

## Prerequisites

Before you begin, ensure you have the following prerequisites installed:

- Node.js (version 20 or higher)
- pnpm (version 9 or higher) or yarn (version 1.22 or higher)
- Git

## Clone the Repository

First, clone the repository to your local machine:

```bash
git clone https://github.com/skaleway/docs-template.git
cd docs-template
```

## Install Dependencies

Next, install the project dependencies:

```bash
pnpm install
```

## Start the Development Server

Finally, start the development server:

```bash
pnpm dev
```

This will start the development server and you can view the project in your browser at [http://localhost:3000](http://localhost:3000).
